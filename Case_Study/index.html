DBMS-Docs

To display the 1st, 2nd, and 3rd position holders (based on Total marks) from the student Table based on following conditions:

  • The student must have scored more than 60% marks in each subject.
  • The student must not have failed in any subject.
  • The student must not be absent for any subject's exam.

Input

       
        SELECT student_id, student_name,science_marks, maths_marks, english_marks,(science_marks + maths_marks + english_marks) 
        AS total_marks,ROUND((science_marks + maths_marks + english_marks)/3, 2) AS average_marksFROM students 
        WHERE science_marks > 60 AND science_marks != 'A' AND maths_marks > 60 AND maths_marks != 'A' 
        AND english_marks > 60 AND english_marks != 'A' ORDER BY total_marks DESC
        LIMIT 3;
