SQL Tutorial: CREATE TABLE Command

Introduction

The CREATE TABLE statement in SQL is used to create a new table in a database. It defines the table name, its columns, and the data types for each column.

Syntax

CREATE TABLE table_name (
    column1 datatype,
    column2 datatype,
    column3 datatype,
    ...
);

Each column must have a name and a data type. Common data types include INT, VARCHAR(n), DATE, and more.

Example

Here's an example of creating a table named Employees:

CREATE TABLE Employees (
    EmployeeID INT PRIMARY KEY,
    FirstName VARCHAR(50),
    LastName VARCHAR(50),
    Email VARCHAR(100),
    HireDate DATE
);

This table includes a primary key and several other fields typically found in an employee database.
